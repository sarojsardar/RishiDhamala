# Rishi Dhamala - Official Campaign Website

Official website for Rishi Dhamala, Janadesh Party Nepal leader and renowned journalist.

## About

This is the official campaign website for Rishi Dhamala, featuring:
- Campaign information and updates
- Biography and achievements
- TV shows and interviews
- Contact and volunteer information
- Election manifesto and promises

## Features

- **Modern Design**: Built with Laravel and Tailwind CSS
- **Responsive Layout**: Mobile-first design approach
- **Election Focus**: Campaign-oriented content and CTAs
- **Media Integration**: Showcase of TV shows and interviews
- **Font Awesome Icons**: Professional iconography throughout
- **Contact Forms**: Easy communication with campaign office

## Technology Stack

- **Framework**: Laravel 11
- **CSS**: Tailwind CSS v4.0
- **Icons**: Font Awesome 6.5.1
- **Build Tool**: Vite
- **PHP**: 8.2+

## Installation

```bash
# Clone repository
git clone https://github.com/sarojsardar/RishiDhamala.git

# Install dependencies
composer install
npm install

# Setup environment
cp .env.example .env
php artisan key:generate

# Run development server
php artisan serve
npm run dev
```
